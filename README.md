1. Se descargan los archivos.
2. En Xampp le colocamos "Start" en Apache y MySQL
3. En archivo en hdocs se coloca el archivo "PikNotes".
4. Abrir admin de MySQL en Xampp, redirigirá a una página. 
5. En la barra izquierda buscamos PikNotes, damos click y luego al boton de SQL, ahí colocamos la bd.
6. El archivo "Example.py" lo abrimos en VScode.
7. En la terminal de Visual colocamos python Example.py
8. Damos click a la ruta y se mostrará nuestro proyecto.
